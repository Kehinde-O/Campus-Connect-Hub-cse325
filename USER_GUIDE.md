## Getting Started

### System Requirements

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- An active internet connection
- JavaScript enabled in your browser

### Accessing the Application

1. Navigate to the Campus Connect Hub URL provided by your institution
2. The application will load automatically in your browser
3. No installation is required - it works directly in your web browser

## Creating an Account

### Student Registration

1. Click the **"Register"** link in the navigation menu
2. Fill in the registration form:
   - **First Name**: Your first name
   - **Last Name**: Your last name
   - **Email**: Your campus email address
   - **Password**: Create a secure password (minimum 6 characters)
3. Click **"Create Account"**
4. You will be automatically logged in after successful registration

### Important Notes

- Use your official campus email address
- Choose a strong password that you can remember
- All new accounts are created as "Student" accounts by default
- If you need administrator access, contact your campus IT department

## Logging In

1. Click the **"Login"** link in the navigation menu
2. Enter your email address and password
3. Click **"Sign In"**
4. You will be redirected to the News Feed upon successful login

### Forgot Password?

If you forget your password, contact your campus IT support for assistance.

## Features Overview

Campus Connect Hub provides four main features:

1. **News Feed** - Stay updated with campus announcements
2. **Events** - Discover and RSVP to campus events
3. **Resources** - Quick access to essential campus services
4. **Admin Dashboard** - For administrators to manage content (admin only)

## News Feed

The News Feed is your central hub for campus announcements and updates.

### Viewing News

1. Navigate to the **News Feed** (home page)
2. News posts are displayed in a card layout
3. Each card shows:
   - Post title
   - Author name
   - Publication date
   - Full content

### Navigation

- Use the **Previous** and **Next** buttons to navigate between pages
- News posts are sorted by date (newest first)
- Only published posts are visible to students

## Events

The Events section helps you discover and participate in campus activities.

### Viewing Events

1. Click **"Events"** in the navigation menu
2. All upcoming events are displayed in a calendar-style layout
3. Each event card shows:
   - Event title and description
   - Date and time
   - Location
   - Current number of attendees
   - Maximum capacity (if applicable)

### RSVP to Events

1. Find an event you want to attend
2. Click the **"RSVP"** button on the event card
3. Your RSVP will be confirmed immediately
4. The attendee count will update to reflect your RSVP

### Canceling an RSVP

1. Find the event you previously RSVPed to
2. Click the **"Cancel RSVP"** button
3. Your RSVP will be removed

### Event Capacity

- Some events have a maximum capacity
- If an event is full, the RSVP button will be disabled
- You can still view full events but cannot RSVP

## Resources

The Resources section provides quick access to essential campus services and links.

### Accessing Resources

1. Click **"Resources"** in the navigation menu
2. Resources are organized by category:
   - Academic
   - Campus Services
   - Support
   - And more

### Using Resources

1. Click on any resource card
2. You will be taken to the external website in a new tab
3. Resources open in a new browser tab so you don't lose your place

### Resource Categories

Resources are grouped into categories for easy navigation:
- **Academic**: Student portal, library, academic resources
- **Campus Services**: Maps, dining, housing
- **Support**: IT help desk, counseling services

## Admin Dashboard

The Admin Dashboard is available only to users with Administrator privileges.

### Accessing the Dashboard

1. Log in with an administrator account
2. Click **"Admin"** in the navigation menu
3. You'll see an overview dashboard with statistics

### Dashboard Features

The dashboard provides:
- **Statistics Overview**: Total news posts, events, RSVPs, and users
- **News Management**: Create, edit, and delete news posts
- **Event Management**: Create, edit, and delete events
- **Resource Management**: Create, edit, and delete resources

### Creating News Posts

1. Navigate to the Admin Dashboard
2. In the News Management section, click **"Create News Post"**
3. Fill in:
   - Title
   - Content
   - Publication status (published or draft)
4. Click **"Save"**

### Creating Events

1. Navigate to the Admin Dashboard
2. In the Event Management section, click **"Create Event"**
3. Fill in:
   - Title
   - Description
   - Event date and time
   - Location
   - Maximum attendees (optional)
4. Click **"Save"**

### Managing Resources

1. Navigate to the Admin Dashboard
2. In the Resource Management section, click **"Create Resource"**
3. Fill in:
   - Title
   - Description
   - URL
   - Category
   - Display order
4. Click **"Save"**

### Editing and Deleting

- Click **"Edit"** on any item to modify it
- Click **"Delete"** to remove an item (this action cannot be undone)

## Troubleshooting

### Common Issues

#### Cannot Log In

- **Problem**: Login fails with "Invalid email or password"
- **Solution**: 
  - Verify your email address is correct
  - Check that Caps Lock is not enabled
  - Ensure you're using the correct password
  - Try resetting your password if needed

#### Page Not Loading

- **Problem**: The page appears blank or doesn't load
- **Solution**:
  - Refresh the page (F5 or Ctrl+R)
  - Clear your browser cache
  - Check your internet connection
  - Try a different browser

#### RSVP Not Working

- **Problem**: Cannot RSVP to events
- **Solution**:
  - Ensure you are logged in
  - Check if the event is full
  - Verify your internet connection
  - Try refreshing the page

#### Cannot Access Admin Dashboard

- **Problem**: Admin link doesn't appear or access is denied
- **Solution**:
  - Verify you are logged in with an administrator account
  - Contact IT support to verify your account permissions
  - Log out and log back in
